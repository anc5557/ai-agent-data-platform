## 핵심 기획 역량

### 1. 비즈니스 문제 정의

- 분산된 고객 이슈를 통합 관리하여 대응 품질 향상
- 팀 간 데이터 단절 해소 및 맥락 유지

### 2. 데이터 아키텍처 설계

- Medallion Architecture (Bronze → Silver → Gold) 적용
- AI 활용을 전제로 한 데이터 구조 표준화

### 3. AI Tool 설계

- 자연어 질의 기반 분석 도구 5종 설계
- 정형 데이터 쿼리 + 벡터 검색 하이브리드 구조

### 4. 성과 측정 체계

- CX (고객 경험), DX (직원 경험), DT (디지털 전환) 관점 정량화
